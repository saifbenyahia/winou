# Manual API Scripts

Ces scripts servent uniquement a des tests manuels locaux.

- Ils ne font pas partie d'une suite de tests automatisee.
- Ils supposent que l'API tourne en local sur `http://localhost:5000`.
- Ils utilisent parfois des comptes de test existants.

Ils ont ete conserves car ils peuvent encore aider au debug ponctuel, mais ils ne doivent pas etre consideres comme la source officielle de verification.
